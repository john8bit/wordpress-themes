Celestial Orbit WordPress Theme
===============================

Created for Celestial Web Development.

INSTALLATION
1. In WordPress admin, go to Appearance -> Themes -> Add New -> Upload Theme.
2. Upload the ZIP file and activate the theme.
3. Go to Settings -> Reading and set your homepage to a static page called Home.
4. Create these pages for the best experience:
   - Home
   - Services
   - WordPress Hosting
   - Portfolio
   - About
   - FAQ
   - Contact
5. Go to Appearance -> Menus and assign your menu to Primary Menu and Footer Menu.
6. Optional: add a widget to the Footer CTA widget area.

WHAT IS INCLUDED
- Custom homepage template via front-page.php
- Responsive layout
- Sticky header and mobile menu
- Blog/page templates
- Footer contact section
- Editable content using standard WordPress pages

RECOMMENDED NEXT STEPS
- Replace the sample portfolio cards with real projects
- Add your contact form shortcode to the Contact page
- Add your logo via Appearance -> Customize -> Site Identity
- Update Settings -> General with your site title and tagline

Celestial Orbit Theme Contrast Light - Production Deployment Package

Included:
- Installable WordPress theme ZIP
- Deployment checklist
- Post-deployment checklist

Deployment steps:
1. Back up the live site files and database.
2. Upload the theme ZIP in Appearance > Themes > Add New > Upload Theme.
3. Activate the theme on staging first.
4. Confirm Settings > Reading points to the correct Home page.
5. Review Appearance > Menus / Navigation.
6. Clear all caches (plugin, server, CDN, browser).
7. Test desktop and mobile layouts.
8. Verify forms, buttons, internal links, and page slugs.
9. Re-save permalinks if needed.
10. Deploy to production only after staging review is complete.

Post-deployment review:
- Homepage alignment
- Header and footer navigation
- Services, About, Hosting, Contact pages
- Performance and caching
- SSL / mixed-content issues
- 404 checks
- Admin editor access and homepage editability
- SEO plugin settings / indexing
